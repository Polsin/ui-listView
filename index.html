ui-listView

An efficient, scrollable, dynamic list view for angular.

View on GitHub Download .zip Download .tar.gz

ui-listView is an efficient, scalable list for large data sets. You simply provide a height and use it similarly to ng-repeat with all the angular bindings you want. Even with a list of 1000 it won't slow down.

Installation

You can add ui-listView through the bower package manager.

bower install ui-listView [--save]

Examples

Basic Usage

{{ country.name }} 


            <div class="contact" ui-list-view="country in countries">
              {{ country.name }}
            </div>

Advanced Usage

{{ contact.name }}
{{ contact.email }}


            <div class="contact-list panel panel-default">
                
                <div class="panel-heading">
                    <input class="form-control" ng-model="search.name" placeholder="Search Name.."/>
                </div>
                
                <div class="panel-body">
                    <div class="contact" ui-list-view="contact in contacts | filter:search">
                        <p class="name"><strong>{{ contact.name }}</strong></p>
                        <div class="email text-muted">{{ contact.email }}</div>
                    </div>
                </div>
                
            </div>
